# Five-Phase Domain Map

Use this file when translating wuxing (wood, fire, earth, metal, water) into non-spatial domains: people, careers, industries, finance, brands, products, organizations, learning, relationships, negotiations, health-adjacent wellbeing, creative work, and life choices.

## Use Rules

- Use five phases as relational processes, not fixed physical substances.
- Diagnose the pattern of the question, not the worth of the person.
- Combine phase symbolism with the domain's real requirements.
- Prefer balancing moves over declaring something good or bad.
- When stakes are high, load `ethics-and-limits.md`.

## Personal Pattern Map

Read patterns in habits and environment:

- Over-fire: too much urgency, screen exposure, speculation, public pressure.
- Over-water: endless research, fear, hidden plans, weak commitment.
- Over-wood: too many projects, no pruning, ambitious but unfocused.
- Over-earth: comfort zone, worry, clutter, over-responsibility.
- Over-metal: perfectionism, harsh self-judgment, excessive control.

## Balancing Moves

| Pattern | Add | Examples |
| --- | --- | --- |
| Over-fire | Water, earth, metal | sleep, cooling schedule, written rules, reserves |
| Over-water | Earth, wood | fixed decision criteria, small action path |
| Over-wood | Metal, earth | prune commitments, define limits, pace growth |
| Over-earth | Wood, metal | movement, declutter, simplify obligations |
| Over-metal | Fire, water | warmth, social repair, flexible review |

Use the smallest effective adjustment first.
