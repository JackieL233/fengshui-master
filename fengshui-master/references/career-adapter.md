# Career Adapter

Use this file when the user asks for feng shui guidance on career direction, promotion, job search, interviews, leadership, work relationships, negotiation, study-to-career transition, or desk/work environment as career support.

## Core Rule

Use feng shui as a symbolic planning and environment-support lens. Do not promise jobs, promotions, income, visas, admissions, or negotiation outcomes.

## Career Intake

Ask for:

- Career goal and timeline.
- Current role, skills, evidence, and constraints.
- Desired industry or function.
- Decision type: choose path, prepare interview, negotiate, improve workspace, time a launch, reduce conflict.
- Support system: mentors, portfolio, references, network, savings.
- Work environment: desk, room, remote setup, commute, team context if relevant.

## Feng Shui Translation

| Feng shui concept | Career translation |
| --- | --- |
| Command position | strategic visibility and ability to see incoming demands |
| Back support | mentor, manager trust, portfolio, savings, credentials |
| Ming tang | visible opportunity, interview pipeline, project runway |
| Sha qi | politics, burnout, unclear role, hostile deadline, public pressure |
| Leakage | scattered applications, weak follow-up, unpaid labor, unclear ask |
| Timing | preparation, application, negotiation, consolidation, rest |

## Five-Phase Career Lens

| Phase | Career expression | Useful action |
| --- | --- | --- |
| Wood | skill growth, new path, long-term plan | build roadmap and portfolio |
| Fire | visibility, presentation, leadership | practice pitch and public proof |
| Earth | reliability, service, operations | stabilize routines and references |
| Metal | standards, craft, negotiation, boundaries | clarify terms and prune distractions |
| Water | research, network, adaptability | informational interviews and market study |

## Decision Pattern

1. **Career reality**: skills, evidence, market, constraints.
2. **Support and visibility**: backing, command, ming tang.
3. **Phase balance**: growth, visibility, stability, rules, research.
4. **Timing**: prepare, ask, apply, negotiate, consolidate.
5. **Actions**: desk/workflow adjustment plus concrete career step.

## Environment Supports

- Prefer a desk with back support and a view of the door.
- Keep the work surface clear enough for the main goal.
- Put proof of work, plan, or active project in a visible but uncluttered place.
- Reduce leakage: notification clutter, unclear task lists, scattered documents.

## Forbidden Claims

Do not claim a direction, color, date, or desk placement guarantees promotion, hiring, salary, or reputation.
